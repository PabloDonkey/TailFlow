# Skill: start-dev-session

Purpose: Resume development quickly by rebuilding project context.

## Steps

1. Read the following files if they exist:

.project/dev-loop.md
.project/tasks.md
.project/roadmap.md
AGENTS.MD
.github/ui-contract.md

2. Summarize:

* Current feature
* Current development step
* Objective
* Current progress

3. Identify the **Next Action** from `.project/dev-loop.md`.

4. Print a short **Development Session Brief** containing:

* Current feature
* Current step
* Files involved
* Progress
* Next action

5. Suggest the **first concrete coding task** to execute next.

## Output format

Development Session Brief

Current Feature:
Current Step:
Progress:
Next Action:

Suggested First Task:
(very concrete implementation step)

## Usage

The developer can trigger this skill by writing:

start-dev-sess
